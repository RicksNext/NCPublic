Changelog - NextCounts v3.1!

New update, new counters!

  • StoryFire Live Subscriber Counts and StoryFire Live Video Views Count have been added!
  • TikTok and Twitch counters have been disabled for now. They will be re-enabled in a future version of NextCounts.
  • For Insiders: the Beta Compare page got a new graph! Now, you can see the average rate per minute of a user in a nice colored graph!
  • TikTok and Twitch have been disabled in the Compare page.
  • YouTube Video Dislikes have been added back to the Analytics! Soon, they will also be present in the counters.


    Bug Fixes:
  • Fixed a bug that would make milestones like 900k show up as 90k. Sorry about that!
  • YouTube Search should work now. It's also worth mentioning that YouTube Video Search is a bit unstable at the moment, and some results are, for some reason, showing up instead of others. When you search up "Rewind 2018" and the actual Rewind video shows up, you'll know we patched it.
